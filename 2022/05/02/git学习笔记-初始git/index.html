git学习笔记-初始git

koderboy
2022-05-02
git › 笔记

Git是目前世界上最先进的分布式版本控制系统。它可以在任何时间点,对文档进行修改保存 也可以在任何时间点对修改的文件进行还原保存。
工作原理 / 流程：

Workspace：工作区

Index / Stage：暂存区

Repository：仓库区/归档区（或本地仓库）

Remote：远程仓库
