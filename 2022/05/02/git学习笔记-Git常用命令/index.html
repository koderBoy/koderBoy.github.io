重置代码

使用 <code> git reset [–mixed|–soft|–hard] <commit> </code> 命令重置本地仓库已提交的代码

  1. mixed

默认不指定参数是使用的是–mixed方式，会重置归档区、暂存区，工作区代码会变回到commit时

  1. soft

使用–soft参数时，会重置归档区，暂存区、工作区会保留

  1. hard

使用–soft参数时，归档区、暂存区、工作区都会被重置
